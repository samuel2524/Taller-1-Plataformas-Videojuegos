# 🎮 Plataforma de Videojuegos

Aplicación web desarrollada con **React + Vite** que permite explorar, registrar y gestionar videojuegos, utilizando **Supabase** como backend para autenticación y base de datos.

---

## 🚀 Características

- 🔐 Registro e inicio de sesión de usuarios
- 🎮 Visualización de videojuegos
- ➕ Agregar nuevos videojuegos
- ✏️ Editar videojuegos existentes
- ❌ Eliminar videojuegos
- 🔍 Filtrado y navegación por categorías
- 📦 Consumo de API externa (RAWG)
- 🧠 Manejo de estado con React Hooks
- 🎨 Estilos con Tailwind CSS

---

## 🛠️ Tecnologías utilizadas

- ⚛️ React
- ⚡ Vite
- 🎨 Tailwind CSS
- 🗄️ Supabase (PostgreSQL + Auth)
- 🌐 API RAWG (Videojuegos)
- 🔀 React Router DOM
